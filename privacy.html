AutoFB - Privacy Policy

Last updated: March 2026

Data Collection

AutoFB does not collect, transmit, or store any personal data on external servers. All data is stored locally in your browser using Chrome's chrome.storage.local API.

What Data is Stored Locally

  • Settings & scenarios — your configured automation preferences
  • Session statistics — counts of likes, comments, etc. for your own tracking
  • Activity logs — timestamped log entries for debugging (max 500 entries)
  • Post tracking IDs — hashed identifiers to avoid duplicate interactions

AI Comment Feature

If you enable the AI comment feature, post content is sent to Google's Gemini API using your own API key. AutoFB does not have access to your API key beyond the local storage in your browser. No data passes through AutoFB servers.

Permissions

  • storage — save your settings and stats locally
  • scripting — inject the automation script into Facebook tabs
  • alarms — scheduled task execution and auto-stop timers
  • host_permissions (facebook.com) — interact with Facebook pages

Third-Party Services

The only external service used is Google Gemini API (optional, user-configured). No analytics, tracking, or advertising services are included.

Data Sharing

AutoFB does not share any data with third parties. All operations happen locally in your browser.
